# BCCL Coal Quality Intelligence System

An AI-powered coal quality analytics platform developed during my internship at Bharat Coking Coal Limited (BCCL).

## Features

- GCV Prediction
- Coal Grade Classification (G1-G18)
- Revenue Estimation
- Batch Analysis
- Industrial Process Recommendations
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Plotly

## Project Structure

- app.py
- model_train.py
- data_generation.py
- Single_data.py
- coal_grade_model.pkl
